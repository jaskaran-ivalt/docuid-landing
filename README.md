# DocuId Landing Page

This is the landing page for the DocuId Word Add-on, featuring biometric authentication integration with iVALT.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Download

**Latest Version:** v1.0.0

Download the standalone installer for Windows 11:

[Download iVALT-Docuid-WordAddin-Installer.exe](https://github.com/jaskaran-ivalt/docuid-landing/releases/download/docuid/iVALT-Docuid-WordAddin-Installer.exe)

---

## Features

- Responsive landing page for DocuId Word Add-on
- Download buttons linked to GitHub releases
- Biometric authentication showcase
- Microsoft Word integration information
- Clean, modern UI with motion effects

## Project Structure

- `src/App.tsx` - Main landing page component
- `src/main.tsx` - Entry point
- `public/` - Static assets
- `index.html` - HTML template

## Build & Deploy

```bash
npm run build
npm run preview
```
